# Examples

This section demonstrates correct and incorrect usage of components and styles.

---

## Correct usage

- Cards used consistently for payment methods
- Primary button used only for the main action
- Icons used alongside text
- States clearly distinguishable

---

## Incorrect usage

- Mixing multiple button styles
- Using icons without text
- Relying on color alone for state
- Adding shadows or elevation
